## ğŸ¨ Quick Tips

### Styling Elements
1. Select any element on canvas

   - **Properties tab**: Change text, links, etc.
   - **Styles tab**: Colors, fonts, spacing, etc.

- Use **Flex layouts** for flexible arrangements

1. Drag **"Form"** block from **Forms** category
